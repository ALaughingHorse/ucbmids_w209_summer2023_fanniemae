Overview

The Fannie Mae Viewer is a web-based tool that provides insights into residential mortgage performance for investors and anyone interested. It has been prepared as part of a UC Berkeley DATASCI 209 Information Visualization class.

On the "Analyze" page, users will find an interactive tool to help gauge the risk of a loan application based on historical residential mortgage data.

The "Explore" page has a variety of visualizations which highlight some trends identified in the dataset.

The Data

The data used in the various visualizations is mortgage performance data covering the last 10 years of conventional, 30-year mortgage loans guaranteed by Fannie Mae (~33% of US mortgage market). Specifically, this project focuses on the reference pool 1474. Fannie Mae provides this data through their Data Dynamics tool, which is freely available to the public.
